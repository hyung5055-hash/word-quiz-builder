## Packages
framer-motion | Smooth animations and page transitions for a premium feel
canvas-confetti | Celebration effect on completing a quiz
@types/canvas-confetti | TypeScript definitions for canvas-confetti
lucide-react | Beautiful icons for the UI

## Notes
- Quizzes are fetched from GET /api/quizzes
- Single quiz with questions fetched from GET /api/quizzes/:id
- Scores and current question state are managed locally in the client during the quiz session
- Using Wouter for routing and Framer Motion for question transitions
